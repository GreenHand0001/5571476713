# 项目简介

## 核心功能
- 基于YOLOv8的药丸检测系统，已训练完成可直接使用
- 支持图片、视频及摄像头进行检测
- 支持自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 支持选择不同检测模型
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 提供完整源码、300张标注数据集、训练好的模型及训练结果
- 包含详细的项目运行教程

## 适用场景
- 药丸质量检测
- 药品分类与识别
- 生产线自动化检测
- 医药研发辅助
- 药品库存管理

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9，集成opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整UI界面源文件及图标
- 配置成功环境可直接运行
- 实时显示目标位置、目标总数、置信度等信息
